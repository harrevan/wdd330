Javascript: Novice to Ninja Chapter 14: HTML5 APIs

The data- Attribute

Way of embeding data on a web page. Ignored by browser, sole purpose is JavaScript use.

Named by developer: ex: data-powers

HTML5 APIs

HTML5 Web Storage

window.localStorage

Geolocation

Obtain geographical position of device. Requires user permission first.

Is a property of navigator object: EX: navigator.geolocation.getCurrentPosition(youAreHere);

Web Workers

Allow processes to run in background, adding concurrency in JavaScript.

Websockets

New protocol allowing 2-way communication with a server, aka push messaging.

Notifications

Displayed evin if web page calling them isn't in current tab.

Requires user permission first.

Multimedia

Allows insertion of audio or video clips into a web page.
