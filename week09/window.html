Javascript: Novice to Ninja Chapter 9: The Window Object

Every JavaScript environment has a global object. The window object is the global object in a browser environment.

The Browser Object Model

Collection of properties and methods containing info about the browser and computer screen.

Going Global

Global variables accessed in all parts of program. Created without const, let, or var keywords. These global variables are properties of global objects (the window object.) See log for example.

Dialogs

Examples: window.alert(), window.confirm(), window.prompt()

Browser Information

window object has properties and methods to provide info about a user's browser.

Which Browser?

window.navigator.user agent

Returns info about user's browser. Can be modified by user, so not reliable.

Location

window.location property contains info about URL of current page.

Browser History

window.history can be used to access info about previously visited pages in current browser session.

Controlling Windows

window.open() opens a new window in the browser. Takes a URL as a parameter. window.close() closes a window

Screen Information

Use window.screen information to gather info about screen, such as height, width in pixles, etc.

Document Object

window objects contain a document object. Document object has properties/methods to deal with page loded into window.

Cookies

Small files saved locally on a user's computer. Used to store information between requests. Used specifically for personalizing browser experience, storing preferences, keeping track of user decisions (shopping cart), authentication...

Take the form of text files with name/value pairs.

Sessions by default, they are deleted once window closes. Can be made persistnet using exires=date.
