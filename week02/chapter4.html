Javascript: Novice to Ninja- Chapter 4: Functions

Defining a Function

3 ways to define functions in javascript

Function Declarations (Function literal)

function hello(){
console.log('Hello World!');
}

Function Expressions

Assigns an anonymous function to a variable.

const goodbye = function(){
console.log('Goodbye World!');
};

Can also be created with a named function

const goodbye = function bye(){
console.log('Goodbye World!');
};

Function Declarations

Declare function using the constructor Function()

const hi = new Function('console.log("Hi World!");');

Variable Numbers of Arguments

Use the rest operator (...) in front of the last parameter in a function declaration to collect all
arguments entered into an array. Helpful if number of arguments taken is unknown

Default Parameters

Used to specify default parameter values in a function

function hello(name='World') {
console.log(`Hello ${name}!);
}
If function is called without an argument, name parameter will be 'World'

Default parameters should ALWAYS come after non-default parameters

Function Hoisting

Hoisting is when Javascript Interpreter moves all variable and function declarations to the top of the current scope.
This allows functions to be invoked before they are defined.
Variable assignment is not hoisted.

Callbacks

Functions that are passed as arguments to another function. Is then invoked inside body of function

Can use anonymous functions as well.

function sing(song,callback) {
console.log(`I'm singing along to ${song}.`);
callback();
}

Array Iterators

forEach()

Loops over array and invoke a callback function on each element in an array.

map()

Similar to forEach(). Creates a new array with results of callback

reduce()

Cumulatively combines results to return a single value. Callback function used to describe how to combine each value of the array with the running total.

[1,2,3,4,5].reduce( (acc,val) => prev + val );
output is 15

filter()

"Returns a new array that only contains items from the original array that return true when passed to the callback"

Chaining Iterators Together

Methods can be chained together on arrays for more powerful transformations.
