Arrays
Best to create an array using array literal: const myArray = [];
Arrays don't have to have the same data types inside. Variety of data types can be used (including another array).
EX: const mixedArray = [ null, 1, [], 'two', true ];
Length property of an array is mutable (can be changed.)

Merging Arrays use concat(['item1', 'item2', etc]) to merge arrays. Merging does not change original array. Use assignment operator to update.
EX: avengers = avengers.concat(['Hulk','Hawkeye', 'Black Widow']);
output: ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow']
An alternative is to use the spread operator (three dots) ...
EX: avengers = [ ...avengers, ...['Hulk','Hawkeye', 'Black Widow'] ];
output: ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow']

Multi-Dimentional Arrays
Flatten Multi-dimensional arrays ising the spread operator (...)

Sets A set is a data structure representing a collection of unique values. NO DUPLICATES
Good for checking if values have been duplicated, faster to check than an array.
All non-primitive values are considered unique, even if they contain the same values.
