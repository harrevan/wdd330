Master Mobile UX Notes

Lesson 1, Step 1: How to Rewire Your Brain for Mobile First UX

>

Design for smallest, least powerful device first and you will focus on content. - Joel Marsh
Whatever you are doing - do mobile first Eric Schmidt
Responsive does NOT equal moblie first. Design for mobile first. Start small, then scale up. Retrofitting from big to small doesn't work as well.
Mobile first is not a fad. It is here to stay.

Lesson 1, Step 2: How to Improve UX by Understanding Users

Observe what users are doing, not always what they are saying.
Be there, be useful, be quick.

Reachability matters. Needs to be considered across mobile devices. (Smartphone vs tablet, vertical vs horizontal, etc.)
Speed matters. even a half second delay can cause traffic drops. Networks matter. 4g vs 3g availiabilty.
Tips:
  Reduce image usage. Use SVG/Webfonts, which are lower bandwidth.
  Optimize and minify CSS and JS files. Remove empty whitespace.
  GZIP files to eliminate unnecessary data.

Lesson 1, Step 3: Better Mobile UX with Strategic Menu Design

Most commonly used menu items should be located at bottom, most easily reachable area.
Placement matters. Improve reachability, creates perception of speed, increases satisfaction.
Hamburger menu vs. "Menu". Just "Menu" wrapped in a box seems more intuitive to most users.

Lesson 1, Step 4: Designing Intuitive and Mobile Friendly Forms

Make it simple so users can use it easily.
Have labels display inside of fields instead of outside.
Make inputs large enough to click easily.
Use float labels.
Make checkboxes large enough to click with a thumb if possible.
When needed, paginate forms, rather than creating a long page. (Use progress indicator)
Enable autocomplete on forms.
Don't hide password by default, but allow users to do so easily if needed.
Check out single line input filed for credit card details (by Brad Frost).
Keyboards matter. Use number pad for number inputs.
Link phone numbers using anchor tag: \a href="tel:12085555555"\

Lesson 1, Step 5: Designing a Better Mobile Advertising User Experience

DON'T USE POP UPS and POP-OVERS!
Make ads small and simple. Allow them to grow when clicked.
Use window between content to show add as scrolling for large ads.
Users visit websites for content, not ads. Content should always come first. Keep it simple and sleek.
