Local Storage Notes

Section tag fills width of page. Can be used to keep elements on the same line.

Local storage uses key-value pairs to store (localStorage.setItem(key, value)) and load (localStorage.getItem(key)) saved data.
From W3 schools: "Local storage objects The localStorage object stores the data with no expiration date. The data will not be deleted when the browser is closed, and will be available the next day, week, or year."

Two examples of setting and getting data from local storage:
1.) //Store:
localStorage.setItem("lastname", "Smith");
// Retrieve
document.getElementById("result").innerHTML = localStorage.getItem("lastname");

2.) // Store
localStorage.lastname = "Smith";
// Retrieve
document.getElementById("result").innerHTML = localStorage.lastname;

To remove an item from local storage: localStorage.removeItem(key);
