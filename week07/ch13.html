Javascript: Novice to Ninja Chapter 13: AJAX

Ajax

Technique allowing web pages to communicate asynchronously with a server. Dynamically updates web pages without reloading.

Clients and Servers

Clients request resources from a server, which processes the request and sends back a response to the client.

APIs

Application programming interface

Collection of methods allowing external access to another program or service.

The Fetch API

Standard for requesting and sending data asynchronously across a network.

Ajax Example

A living standard

Fetch API spec is still being developed.

Libraries exist to help take care of Ajax requests.

Ex: jQuery has ajax() method used similarly to fetch:

$.ajax('http://numbersapi.com/random').done(text => outputDiv.innerHTML = text );
