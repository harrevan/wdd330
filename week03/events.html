JavaScript: Novice to Ninja Chapter 7:Events

Event Listeners

Event listeners are added to elements, are part of DOM. EX: document.body.addEventListener("click", doSomething);

Inline Event Handlers

Original way of dealing with events. Not recommended for the following reasons:
JavaScript should be kept out of HTML.
Only one event handler for each event-type can be attached to an element.
Event handler code hidden away in markup, making identification difficult.
JavaScript entered as string, making quotes a problem at times.

Recommended: Use EVENT LISTENERS
EX: document.body.addEventListener('click',doSomething);
Parentheses excluded. If included, function is called when event listener is set.

The Event Object

When event handler is triggered by an event, the callback function is called.

Event Propogation

Clicking one event clicks all elements it is nested inside of.

Event propogation is the order that the events fire on each element. 2 types:

Bubbling- when event fires on the element clicked on first, then bubbles up document tree firing event on each parent event.

Capturing starts by firing an event on the root element, then propogates downwards.
