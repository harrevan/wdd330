JavaScript: Novice to Ninja Chapter 6: The Document Object Model

DOM allows for access to elements of a web page. It represents an HTML document as a network of connected nodes forming a tree-like structure
DOM treats everything on the web page as a node, indluding HTML tags, text, attributes, etc. HTML tag is root node.
DOM is not a part of javascript, it is language agnostic. any programming language can use it.
Javascript accesses DOM using the document object.

Getting Elements

Legacy DOM Shortcut Methods

document.body - returns body element
document.images - returns node list of images
document.links - returns node list of 'a' elements and 'area' elements with an href attribute
document.anchors - returns node list of 'a' elements that have a name attribute
document.forms - returns a node list of all forms

Getting an Element by its ID

getElementById() method returns a reference to the element with the specified id

Get Elements by Tag Name

Returns a live node list of all elements with the tag name provided.

Get Elements by Class Name

getElementsbyCLassName() returns node list of all elements with specified class name

Query Selector

document.querySelector() allows use of CSS notation to find first element in document that matches a CSS selector.

document.querySelectorAll() allows use of CSS notation but returns list of all elements matching CSS query selector.

querySelector() can be used on any element, not just document.
