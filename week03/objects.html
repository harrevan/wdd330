JavaScript: Novice to Ninja Chapter 5: Objects

Object Literals

Used to keep related information and functionality together.

Properties are key-value pairs that are separated by commas.

All objects are mutable, even when declared const. Properties and methods can be changed or removed, or added to the object.

Accessing Properties

2 ways to access properties, with dot notation, or bracket notation.

EX: superman.name OR superman['name']

Finding all the Properties of an Object

Use for in loop to find all of an objects properties and methods.: for(const key in superman){console.log(key + " " + superman[key])}

Object.keys() and Object.values() methods returns arrays of objects keys and values respectively.

Adding Properties

A property can be added to an object at any time by simply assigning a value to the new property. EX:
superman.city = 'Metropolis'; \\this statement adds the property city to the object superman

Removing properties

Properties can be removed using delete operator. Ex: delete superman.fly

Nested Objects

Objects can be nested. (Object contains an object)

Namespacing

Namespacing can be used to avoid naming collisions (same variable or function names used for different purposes.) Using an object literal can solve this.

Object literal serves as namespace.
